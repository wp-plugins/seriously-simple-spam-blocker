== Description ==

A plugin that interacts with the StopForumSpam.com api to protect your
website. Find out how many unwanted attempts have been blocked in Dashboard >
Tools > Seriously Simply Spam Blocker.

== Usage ==

Simply upload/activate the plugin and you're good to go.

== Installation ==

Installing "Seriously Simple Spam Blocker" can be done either by searching for "Seriously Simple Spam Blocker" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Customise the plugin from its settings page
